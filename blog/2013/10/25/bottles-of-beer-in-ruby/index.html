Bottles of Beer in Ruby

Oct 25th, 2013

The “99 Bottles of Beer on the Wall” seems like a common exercise that many people use in many different programming languages. I got held up on this for a while until I went through it with a classmate and we broke down the problem into smaller and more manageable chunks. I find that most of the time my biggest problem is where to start. What I have found to be really helpful is to make a list of what the program should do and then stating it in simple terms. For example, in the Bottles of Beer homework this is what we wanted to see happen:
1. we need each loop to show the number of bottles – then each reduced by one
2. we need print bottles of beer on the wall
3. we need the last line to change to no more bottles of beer
4. we need for bottles to be bottle in the case of one bottle
5. we need very last verse to be entirely different when it complete
